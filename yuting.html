＊Git 操作筆記＊

個人小小整理

第一步：安裝

  1. 點選這裡
  2. 按下 Download 吧！：）

第二步：使用者（初次）設定

  1. 打開你的 Terminal → Terminal 相關指令介紹
  2. 輸入以下指令，OOO部分自行修改
    3. $ git config --global user.name "OOOOOOO"
    $ git config --global user.email "OOOOO@OOOOOOO"
  3. 確認一下自己的設定
    4. $ git config --list
  4. 設定別名(依照個人喜好，也可不做XD)
    5. $ git config --global alias.'別名(alias)' '指令'

第三步：新增 Repository

全新開始～

  1. 建立一個要做專案的資料夾並 $ cd 切換過去
  2. 輸入以下指令，進行初始化（讓 Git 對這個目錄開始版控）
    3. $ git init
    會在該目錄中建立 .git 資料夾

本來就有專案～

  1. 直接$ cd 切換到專案目錄
  2. 同上

從 Github 上抓～（Homework都是用這個！！！）

  1. 直接 $ cd 切換到要進行專案的目錄
  2. 輸入以下指令，進行clone
    3. $ git clone 'url'
    會在該目錄中建立一個該 Repository 的資料夾

第四步：進行 Git 專案控管

簡易概念圖


常用指令

指令 功能說明
$ git clone 複製儲存庫
$ git init
(initiate)		 初始化版本庫
$ git status 當前的 Git 狀態(不時就打一下這個指令做確認)
$ git add 加入追蹤(tracking)
$ git rm
(remove)		 刪除檔案
$ git commit 提交目前暫存區的修改，會產生log紀錄
$ git log 查看所有歷史記錄
$ git push 提取本地儲存庫的檔案並推送至遠端儲存庫
$ git branch 列出分支

指令詳細介紹

  1. $ git clone
    • $ git clone 'url'

      • 將url(網址)的遠端儲存庫複製到自己的電腦，目錄裡會產生一個同名資料夾
    • $ git clone 'url' 'name'

      • 複製到自己的電腦，並將資料夾命名為name
  2. $ git add
    • $ git add 'file'

      • 將file(資料夾名稱)加入追蹤
    • $ git add -u

      • 只將修改過的(edited)檔案加入追蹤
    • $ git add --all或是$ git add .

      • 將所有檔案加入追蹤
    • $ git add -i

      • 進入互動模式，依照指示操作唄
  3. $ git commit
    • $ git commit

      • 將目前暫存區的修改進行提交，進入編輯 message(打上提交的內容備註，會顯示在log記錄)階段
    • $ git commit -m 'message'

      • 直接加上message一起提交，跳過編輯階段
  4. $ git remote
    • $ git remote (show)

      • 顯示遠端儲存庫名稱(預設為origin)，show可省略
    • $ git remote -v

      • 顯示遠端儲存庫名稱及位置
  5. $ git push
    • $ git push

      • 將本地分支(預設為master)已提交的(committed)檔案推送至遠端儲存庫分支(預設為origin)
    • $ git push 'remote' 'branch'

      • 將本地分支(branch)的檔案推送至遠端儲存庫分支(remote)
      → 意即$ git push的作用等於$ git push 'remote' 'branch'
  6. $ git branch
    • $ git branch

      • 列出本地分支
    • $ git branch -r

      • 列出遠端分支
    • $ git branch -a

      • 列出所有分支
    • $ git branch -v

      • 列出每個分支最後的提交
